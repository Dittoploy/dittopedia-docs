# Dittopedia

Bienvenue dans la documentation technique de **Dittopedia**.

## Qu'est-ce que Dittopedia ?

Dittopedia est un jeu de devinettes Pokémon. Le joueur doit identifier un Pokémon
à partir d'indices progressifs (talents, types, stats, génération, etc.).

## Stack technique

| Composant | Technologie |
|-----------|------------|
| Frontend | Next.js 16 + React 19 + Tailwind CSS |
| Backend | NestJS 11 + TypeScript |
| Base de données | PostgreSQL 17 |
| Cache / Sessions | Redis 7 |
| API externe | [PokéAPI](https://pokeapi.co/) |
| Documentation | Nextra |
| CI/CD | Jenkins + EC2 Fleet |
| Conteneurisation | Docker |
| IaC | Terraform + Ansible |
